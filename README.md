# AI Resume Screener & Matcher

An Automated Applicant Tracking System (ATS) backend API designed to screen resumes against job descriptions using generative AI. 

## 📝 About the Project
* **PDF Extraction:** Automatically extracts text from uploaded PDF resumes in real-time.
* **AI Analysis:** Compares candidate resumes with job requirements to calculate match percentages and identify missing skills.
* **Data Persistence:** Automatically logs and tracks candidate profiles and resume data.

## 🛠️ Tech Used
* **Framework:** .NET 8 Web API (C#)
* **AI Engine:** Google Gemini 2.5 Flash API
* **Database Engine:** SQLite
* **ORM:** Entity Framework Core
* **Library:** PdfPig (for PDF parsing)
